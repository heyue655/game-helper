# 和悦电竞系统需求

## H5页面

### 框架

使用H5开发，支持常见手机尺寸，如iphone、华为、小米、vivo、oppo等

使用技术mysql + redis + node + react，需要使用阿里云oss存储图片

底部共4个菜单：首页、专区、我的

### 首页

**搜索** 需要支持搜索，按照商品名称搜索，顶部搜索框，在搜索框旁边需要有筛选功能，按照游戏名称筛选（游戏列表从接口获取）

**BANNER** 顶部搜索框下面有一个BANNER区，最多3张可以轮播

**商品列表** 采用卡片形式展示已上架的商品，需要展示缩略图、商品名称、销量、浏览量、现价、原价，原价需要使用中划线，商品列表需要可以分页滚动展示，每次滑动展示6条数据

**商品详情** 展示商品的图片和详细信息，分区块展示

* 区块一：商品名称、描述、现价、原价、收藏

* 区块二：商品规格和规格选择
* 区块三：商品详细图片和文本
* 区块四：通用平台说明（所以的商品详情页默认展示此块内容）

**详情操作** 购买、联系客服

- 购买：用户下单后处于待支付状态，支付成功后为待分配状态，管理员可以在后台分配接单人、分配完成后处于待交单状态，**接单人在 H5「我的任务」中点击「确认交单」（需二次确认弹窗）**，交单后订单直接变为已完成状态（无需下单人二次确认）。管理后台不提供交单按钮，交单操作仅由 H5 接单人执行。

  > 订单状态流转：待付款 → 待分配 → 待交单 → 已完成 → 待结算 → 已结算
  >
  > 特殊状态：已关闭（待付款订单30分钟内未支付自动关闭，不计入任何统计）

- 待付款订单进入详情后展示30分钟倒计时，超时未支付订单状态自动变为"已关闭"，关闭订单不计入角标数量统计。

- 支付：目前采用沙箱模式模拟支付，支付宝真实对接信息待接入后替换。

**下拉选择器** 所有页面的下拉选择均使用自定义底部弹出组件（BottomSheetPicker），避免原生 select 在移动端样式异常，弹层使用 ReactDOM.createPortal 挂载到 body，不受父级层级影响。



### 专区

专区及各种商品的分类，包含：性价比专区、保底专区、清图专区、活动专区、大红专区、带出专区、保险专区、金牌考核专区。

切换不同专区展示不同的商品即可，需要滑动分页展示，每次滑动加载6张图片即可



### 我的

**BANNER背景** 这个区域显示个人头像和背景，昵称等，支持修改

**订单区** 共全部、待付款、待分配、待交单4个横排展示的菜单，分别展示不同状态的订单；待付款、待分配、待交单有数量时显示红色角标（已关闭订单不计入角标）

**功能区** 

- 我的任务：展示分配给本人的"待交单"与"已交单"订单（按进行中/已完成分组展示），如有待交单任务显示红色角标；点击可以查看订单详情，支持点击"确认交单"按钮（需二次弹窗确认），交单后订单直接变为已完成
- 我的收藏：展示本人收藏的商品
- 我的投诉：对"已完成"的订单可以发起投诉，填写投诉原因，关联订单的"是否被投诉"字段

## PC端

支持账号密码登录

**账号体系**

- 超级管理员（SUPER）：默认初始化账号为 admin / admin123，拥有所有菜单和功能权限
- 管理员（ADMIN）：由超级管理员在系统管理中创建，登录后无首页看板和系统管理权限，其余功能菜单正常可用
- 菜单顺序：首页看板、订单管理、商品管理、玩家管理、游戏管理、专区管理、Banner 管理、系统管理
- 非 SUPER 角色登录后默认跳转到订单管理页

**首页看板**（仅超级管理员可见）

展示今日订单总额、累计订单总额、今日待交单、今日待结算、累计已交单、累计已结算

- 今日订单总额：展示今日除"待付款"和"已关闭"外的订单累计金额
- 累计订单总额：展示除"待付款"和"已关闭"外的所有订单累计金额
- 今日待交单：统计今日已分配但未完成的订单数量
- 今日待结算：展示今日已分配但未完成的订单总额
- 累计已交单：展示所有已完成+待结算+已结算状态的订单数量
- 累计已结算：展示状态为"已结算"的订单总额（不含已完成、待结算）

**商品管理** 

- 商品列表：游戏名称、商品名称、现价、原价、销量、浏览量、上架时间、操作(上架/下架/删除)
- 上架：状态改为已上架
- 下架：将已上架商品修改为下架
- 删除：仅下架商品支持删除，删除需要二次确认
- 新增：支持新增商品（游戏列表从接口获取）

**订单管理** 

订单状态：待支付、待分配、待交单、已完成、待结算、已结算、已关闭

- 订单列表：游戏名称、商品名称、玩家昵称、交接人、价格、状态、下单时间、是否被投诉、操作（分配、结算）
- 点击分配后弹出系统内置玩家，将订单分配给玩家，对应订单表中的"交接人"
- 被投诉的订单可以在列表中点击查看投诉原因和投诉时间
- 订单状态变为已完成且两小时内没有投诉时，定时任务自动将订单状态改为待结算（每5分钟检查一次）
- 点击结算按钮后订单变为已结算状态

**玩家管理** 

- 玩家列表：头像、昵称、手机号、性别(男|女|未知)、年龄、游戏、是否内置玩家、操作(拉黑)
- 拉黑后无法下单和评论

**游戏管理**

- 维护游戏列表，目前支持：三角洲、王者荣耀、英雄联盟、原神、和平精英
- H5首页筛选和后台新增商品均从此接口获取游戏列表

**专区管理**

- 维护专区列表，支持新增/编辑/删除，字段包含专区名称、图标、排序

**Banner 管理**

- 维护首页 Banner 图片

**系统管理**（仅超级管理员可见）

- 管理员列表：展示所有管理员账号，支持新增、删除
- 超级管理员可创建普通管理员（ADMIN），普通管理员无法访问此功能
